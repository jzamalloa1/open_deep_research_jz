## Contributing

When documenting a new improvement:

1. Create a new markdown file with a descriptive name (e.g., `feature-name.md`)
2. Follow the template structure from existing improvements
3. Add an entry to this INDEX.md file
4. Update the category quick links if needed
5. Include the date, status, and brief description
